# 📚 AI Study Planner

AI Study Planner is a React-based web application that helps students create personalized study schedules based on their exam date, topics, and daily study hours.

## 🚀 Features

* Generate a day-wise study plan
* Distribute topics automatically across available days
* Track study progress with checkboxes
* Progress percentage indicator
* Input validation for exam dates and study hours
* Local Storage support (data persists after refresh)
* Responsive and user-friendly interface

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript
* Local Storage API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Srikeerthi469/AI-Study-Planner.git
```

Navigate to the project folder:

```bash
cd AI-Study-Planner
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## 🎯 How It Works

1. Enter the subject name.
2. Select the exam date.
3. Add topics (one per line).
4. Enter daily study hours.
5. Click **Generate Study Plan**.
6. Track your progress by marking completed tasks.
